# redteam/reports

This directory stores generated red-team run outputs produced by CI or local evaluation runners.

Recommended report naming:
- `orchestrator_eval_<date>_<suite_version>.json`
- `renderer_eval_<date>_<suite_version>.json`
- `e2e_eval_<date>_<suite_version>.json`

Keep generated reports out of long-lived source control unless they represent approved baselines.
